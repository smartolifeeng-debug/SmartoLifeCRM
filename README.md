# Smarto Life CRM

Offline-first Windows desktop CRM for Smarto Life.

Current version: `v0.3.0`

## Current Milestone

Version `v0.3.0` is a professional CRM shell milestone focused on UI and architecture only.

Included:

- .NET 8 WPF desktop shell.
- MVVM navigation.
- Dependency injection.
- Navigation, theme, configuration, logging, and error handling services.
- SQLite startup initialization.
- Fluent-inspired Smarto Life theme.
- Collapsible animated sidebar.
- Top toolbar, global search, notification icon, user profile area, and status bar.
- Placeholder pages for Dashboard, Contacts, Products, Quotations, Calendar, Reports, and Settings.

Not included:

- GPS, Maps, WhatsApp, QR, OCR, Reports engine, Backup implementation, Import/Export, AI, Authentication, or User Permissions.

This project is initialized for a phased build:

- Phase 0: blueprint, architecture, schema, module mapping, UI structure
- Phase 1: unified contacts, dashboard, category, tags, business matter, search, settings
- Phase 2: products, quotations, GPS, map view
- Phase 3: WhatsApp marketing, QR scanning, OCR scanning
- Phase 4: backup and restore, recycle bin, audit log, optimization, polish
- Phase 5: users, permissions, activity timeline, attachments, import/export, reminders, security, company profile, notifications, consent, duplicate merge, onboarding, deployment readiness
