# Smarto Life CRM

Offline-first Windows desktop CRM for Smarto Life.

This project is initialized for a phased build:

- Phase 0: blueprint, architecture, schema, module mapping, UI structure
- Phase 1: unified contacts, dashboard, category, tags, business matter, search, settings
- Phase 2: products, quotations, GPS, map view
- Phase 3: WhatsApp marketing, QR scanning, OCR scanning
- Phase 4: backup and restore, recycle bin, audit log, optimization, polish
- Phase 5: users, permissions, activity timeline, attachments, import/export, reminders, security, company profile, notifications, consent, duplicate merge, onboarding, deployment readiness
