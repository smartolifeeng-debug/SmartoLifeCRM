# Smarto Life CRM UI Structure

## Visual Direction

- Primary color: orange.
- Secondary color: black.
- Background: white.
- Style: clean, premium, modern, professional, readable, and business-ready.
- Layout: desktop-first and touchscreen-friendly.
- Interaction: large buttons, clear spacing, simple forms, minimal typing, fast search, quick filters, and guided workflows.

## Main Navigation

Primary navigation:

- Dashboard
- Contacts
- Products
- Quotations
- Marketing
- Map
- Scans
- Reports
- Settings

## App Shell

The app shell should include:

- Smarto Life logo area
- Left navigation rail or sidebar
- Global search
- Notification icon
- Current user/profile menu
- Quick add action
- Backup status indicator
- Main content area

## Dashboard

Dashboard widgets:

- Total contacts
- Customers
- Suppliers
- Products
- Quotations
- Campaigns
- Recent activity
- Map summary
- Backup status
- Quick add buttons
- Search shortcut
- Recent records

## Unified Contact Screens

Contact list modes:

- List view
- Grid view
- Map-linked view
- Recycle bin view
- Duplicate review view

Contact form sections:

- Identity
- Roles
- Category
- Tags
- Business matters
- Communication
- Address
- GPS
- Consent
- Ownership
- Notes
- Attachments
- Activity timeline
- Custom fields

## Touchscreen Rules

- Favor large tap targets.
- Avoid dense table-only workflows.
- Provide quick action buttons for common tasks.
- Use chips for roles, tags, categories, and matters.
- Use segmented controls for common filters.
- Use review screens after QR/OCR/import automation before saving.
