# Smarto Life CRM v0.3.0 Release Notes

## Release Type

Professional CRM Shell milestone.

This release focuses only on UI and architecture.

## Added

- Full-screen responsive WPF desktop shell.
- Modern Fluent-inspired interface.
- Collapsible animated sidebar.
- Top toolbar with global search, notification icon, settings shortcut, and user profile area.
- Status bar.
- MVVM navigation service.
- Navigation pages:
  - Dashboard
  - Contacts
  - Products
  - Quotations
  - Calendar
  - Reports placeholder
  - Settings
- Dashboard with sample KPI cards, recent activity, upcoming tasks, quick actions, recent customers, recent quotations, placeholder charts, and today's summary.
- Settings placeholders:
  - Theme
  - Company Profile
  - Database
  - Backup
  - Application Information
- Theme service.
- Configuration service.
- File logging service.
- Central error handling service.
- SQLite startup initialization retained.

## Improved

- Smarto Life branding.
- Typography and spacing.
- Shared theme resources.
- Reusable styles for buttons, cards, navigation, DataGrid, TextBox, ComboBox, dialog cards, and tooltips.
- Keyboard focus indicators.
- Touch-friendly control sizes.

## Not Included

- GPS
- Maps
- WhatsApp
- QR
- OCR
- Reports engine
- Backup implementation
- Import/export
- AI
- Authentication
- User permissions
- Contacts module implementation
